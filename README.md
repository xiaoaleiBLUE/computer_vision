# 三、手掌关键点检测
>- 1. opencv 读取摄像头视频流，并在窗口进行实时显示
>- 2. 在读取的画面上绘制一个方块
>- 3. 通过 mediapipe 包获取手指关键点坐标，将这些关键点进行绘制
>- 4. 判断手指上的某一个关键点是否在方块上
>- 5. 如果该关键点在方块上, 方块跟着手指移动（手上的关键点动，方块随之跟着移动）
